CloudFlare bypass, Breacher, FindPerson, GeneratorIP

Dependencies: **libcurl**
## Linux
- git clone https://github.com/Lomasterrrr/HackWebTool.git
- cd HackWebTool-main
- make

      FindPerson:
          Usage: ./HackTool [-FP] [-n, --name]
      GeneratorIP4:

    optional arguments:
